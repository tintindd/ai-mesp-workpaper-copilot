# AI-MESP Workpaper Copilot

面向 MESP 底稿的智能取数、自动重算与异常复核助手。

当前项目包含：

- `mesp_copilot_frontend.html`：前端演示页面
- `scripts/`：训练资料分析与自动化处理脚本
- `training_files/`：MESP/SAP 示例训练文件
- `mesp_training_profile.json`：训练资料结构化结果
- `mesp_automation_result.json`：自动化处理结果示例
- `training_files_inventory.json`：训练文件清单

项目目标是支持 MESP 底稿中的 SAP 证据读取、字段映射、自动重算、异常识别、证据追溯和底稿输出。
